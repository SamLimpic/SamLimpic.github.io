Fellowship

Full-Stack: Vue3 - SCSS - C#

Completed In: 40 Hours

Team Size: Individual

Team Role: Full-Stack

  • A site inspired by Pinterest, populated with "users" from The Lord of the Rings, with authentication through Auth0.
  • Allows multiple user profiles, user-submitted "Keeps", and user "Vaults".
  • Users can post Keeps, and save any user's Keeps to a personal Vault.
  • A user's keeps and public vaults can be viewed on their public profile page.
  • Posts are tagged, and users can search for Keeps by tag name.
  • User profiles, keeps, vaults, and tags are all stored in a MySQL database.

The Tavern

Full-Stack: Vue3 - SCSS - Node.js

Completed In: 80 Hours

Team Size: Four Developers

Team Role: Product Owner

  • An interactive character creation app for Dungeons and Dragons, with authentication through Auth0.
  • Designed as an entry point for users with little knowledge of the game.
  • Allows multiple user profiles, and stores all user-generated Characters.
  • Users complete a personality profile to determine their ideal character.
  • Pulls data from a custom-built MongoDB Database to build characters according to the official D&D ruleset.
  • Produces a fully-playable starting character that can be referenced from a user's character profile page.

Bug Catcher

Full-Stack: Vue3 - SCSS - Node.js

Completed In: 20 Hours

Team Size: Individual

Team Role: Full-Stack

  • A tech site inspired by StackOverflow, with authentication through Auth0.
  • Designed as a portal for user-submitted software "Bugs", allowing users to comment until Bugs are closed.
  • Modified to instead allow users to "Catch" and "Release" Bug Pokemon.
  • Data is pulled from the Pokemon API and displayed on a Bug's info page.
  • Users may "Release" bugs, but their data remains in a MongoDB database to be viewed publicly by all users.

Bill Boards

Full-Stack: Vue3 - SCSS - Node.js

Completed In: 40 Hours

Team Size: Two Developers

Team Role: Front-End

  • A task manager inspired by Trello, with authentication through Auth0.
  • Users can create "Boards" full of "Lists", which contain "Tasks", utilizing embedded Vue components.
  • Tasks can be commented on and removed when complete.
  • Tasks can be moved between lists utilizing HTML Drag-and-Drop.
  • Boards are tied to a user's account, stored in a MongoDB database.
